PEmbroider

A Library by The Frank-Ratchye STUDIO for Creative Inquiry for the Processing programming environment.
Last update, 10/07/2020.

A library for generating .DST, .PEC, .VP3, and other files for computer-controlled embroidery machines.
Intelligibly structured to resemble Processing's own rendering commands, this library offers a wide range of methods of filling, overlapping, cropping, and merging shapes. Makes extensive use of computational geometry algorithms to optimize stitch placement.

Download

Download PEmbroider version 1.0.0 (1) in .zip format.

Installation

Unzip and put the extracted PEmbroider folder into the libraries folder of your Processing sketches. Reference and examples are included in the PEmbroider folder.
